# 🚀 AI Codebase Analyzer

AI-powered multi-agent platform for repository analysis, semantic code search, documentation generation, and architecture visualization.

## ✨ Features

* Repository upload via ZIP or GitHub URL
* AI-powered code analysis
* Semantic code search
* Automated documentation generation
* Architecture visualization
* Multi-agent orchestration

## 🛠️ Tech Stack

* FastAPI
* Streamlit
* OpenAI API
* ChromaDB
* SQLite
* Python

## 📌 Future Improvements

* GitHub OAuth
* Multi-language support
* Advanced analytics dashboard
